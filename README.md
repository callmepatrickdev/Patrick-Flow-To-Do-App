# Patrick Flow 🎯

Patrick Flow is a modern, minimalist, and stress-free To-Do web application designed to help users structure their day with clarity. By separating urgent daily tasks from future backlogs, it removes cognitive overload and keeps you focused on what matters most.

🔗 **[Live Demo](https://callmepatrickdev.github.io/To-Do-List-/)**

---

## 🚀 Features

*   **Smart Task Organization:** Categorize tasks into **Focus Today** or **Inbox / Later** to keep your current view clutter-free.
*   **Minimalist Workflow:** The task creation panel is cleanly tucked away inside a modal popup, keeping your dashboard beautifully clean.
*   **Context-Aware Controls:** The "+ Add New Task" option automatically adapts, showing only when you are on your active focus view.
*   **Celebration Micro-interactions:** Instant "Hurray!" banner notifications animate on-screen to celebrate your wins the moment a task is completed.
*   **Dedicated History Tab:** A clean **Completed** tab keeps a record of all your achievements.
*   **Persistent Storage:** Built-in LocalStorage integration ensures your data is never lost, even if you refresh or close the browser.
*   **Dynamic Empty States:** Delightful visual states guide you when a dashboard list is entirely clear.

---

## 🛠️ Tech Stack

This project is built purely with standard web technologies, requiring no external frameworks or dependencies:

*   **HTML5:** Semantic structure and accessible modal interfaces.
*   **CSS3:** Responsive layout utilizing flexbox, grid systems, custom CSS variables, and clean animations.
*   **JavaScript (ES6+):** Vanilla script handling reactive UI rendering, tab navigation transitions, and data persistence state logic.

## ⚡ Getting Started

To run this project locally on your machine:

1. **Clone the repository:**
   ```bash
   git clone https://github.com
   ```
2. **Navigate into the directory:**
   ```bash
   cd To-Do-List-
   ```
3. **Open the application:**
   Simply double-click the `index.html` file to launch it instantly in any modern web browser.
